# Changelog - jupiter/web/js/auto_form.js

## v0.1.0
- Initial creation of AutoForm module
- JSON Schema-based automatic form generation
- Support for all JSON Schema primitive types (string, number, integer, boolean)
- Array and object type handling with nested forms
- Enum fields rendered as select dropdowns
- Form validation based on JSON Schema constraints
- Required field handling and visual indicators
- Conditional rendering via `if`/`then`/`else` schema
- Live data binding with onChange callbacks
- CSS styling matching Jupiter dark theme
- Error message display for validation failures
- Support for format hints (date, email, uri, textarea)
- Nested object and array editing UI
