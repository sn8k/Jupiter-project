# Changelog - jupiter/web/js/logs_panel.js

## v0.1.0
- Initial creation of LogsPanel module
- Real-time log streaming via WebSocket
- Log level filtering (DEBUG, INFO, WARNING, ERROR, CRITICAL)
- Text search with highlighting
- Timestamp formatting options (relative/absolute)
- Auto-scroll with pause on user interaction
- Log entry color coding by severity
- Copy log entry to clipboard
- Clear logs functionality
- Export logs as JSON or plain text
- Source filtering (core, plugins, server, etc.)
- Timestamp-based ordering
- Maximum log buffer with auto-trim
- Performance optimized with virtual scrolling hints
